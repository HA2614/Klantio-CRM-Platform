# Klantio-CRM-Platform
Klantio is een CRM platform voor freelancers en opdrachtgericht werk. Het combineert opdrachtbeheer, facturatie, PDF factuurscans, urenregistratie en maandelijkse omzetoverzichten in één webapplicatie. Gebouwd met Node.js, Express en PostgreSQL, met Auth0 authenticatie en strikte data scheiding per gebruiker.
